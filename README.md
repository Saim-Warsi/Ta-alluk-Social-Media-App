##  Ta'alluk - A social media app

A modern full-stack social media application built with the MERN stack.

## Features

- Feed - Add your own and browse posts and stories of other users with imageKit.
- User Authentication - Secure login/registration with Clerk.
- Connection Requests - Build ta'alluks by sending follow requests.
- Messaging - Real time messaging conversation.



 ## Tech Stack
- Frontend
    - React.js  | Tailwind CSS
- Backend
   - Node.js | Express.js | MongoDB
- Auth
   - JWT | Bcrypt
- Event-driven workflow automatio
   - Inngest
- Media optimization and delivery
   - ImageKit
